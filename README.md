## About Me

I've been a full-stack engineer for 15 years. Most of that time went into production web apps: banking dashboards at **FIS**, MSP event pipelines and REST APIs at **Kaseya**, dataset catalogs and platform tooling at **NVIDIA**. Right now I'm at **Invisible Technologies**, checking whether model-written code and answers hold up before they go into training or assessment workflows.

I get the most satisfaction from work that looks dull on a slide deck until it breaks. Correct APIs. Postgres schemas that match how the data actually behaves. Code that runs in a sandbox, not just reads well on paper. Dashboards that finish before you open another tab.
